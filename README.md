# Watch Store

## A complete E-commerce template with 100% Test Coverage.

> ## Libraries and Tools

* NPM
* JavaScript
* Vue.js
* Nuxt
* Jest
* Cypress
* Mirage.js
* Eslint
* Prettier
* TailWind

> ## Tests Features

* Unit Tests
* Integration Tests
* E2E Tests
* Mocks
* Test Doubles
* Test Coverage

> ## How to run

- Download project:
```bash
git clone git@github.com:DionerWeiss/watch-store.git
```
- Install dependencies:
```bash
npm ci
```
- Run project:
```bash
npm run dev
```
- Run unit and integration tests:
```bash
npm run test:watch
```
- Run e2e tests:
```bash
npm run test:e2e
```
